# Similarity-Aware-Label-Smoothing

## Abstract
Label smoothing is a widely used regularization technique for reducing overconfidence in neural networks, improving generalization and calibration for multi-class classification. However, conventional uniform label smoothing  distributes probability mass indiscriminately across non-target classes, assuming equal severity for all misclassifications and producing increasingly diffuse targets that yield diminishing per-class gradients as the number of classes grows. To address these limitations, we learn a latent space representation and use inter-class distances to identify semantically related classes. Label smoothing redistribution is then restricted sparsely to a small neighborhood of the target class, with soft targets weighted by distance to focus on plausible misclassifications. Experiments demonstrate improved calibration, as measured by the expected calibration error, and suggest that sparsity is not a secondary design choice but a central ingredient for effective label smoothing.
